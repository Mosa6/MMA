## サイト概要
### サイトテーマ
<総合格闘技のジムを検索、会員登録後コメント、評価できるサイトです。>

### テーマを選んだ理由
<格闘技が好きでその中でも打投極の要素を持つ総合格闘技が好きなので、総合格闘技のジムだけにフォーカスしたサイトがあればどんなジムが自分に合ってるか判断しやすいと思ったからです>

### ターゲットユーザ
<総合格闘技を習いたいと思っている人向けです>

### 主な利用シーン
<総合格闘技ジムを検索でき、会員登録を行えばコメント、評価ができます>
